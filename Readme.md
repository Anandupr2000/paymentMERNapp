# Introduction
This is payment system built using MERN stack for seamless transaction online/offline

# Installation
install project dependencies
1) Client

```
bash
cd client 
npm i
```

2) Server
```
cd server
npm i
```
# Running
1) Client
```
cd client
npm start
```
2) Server
```
cd server
```
```
npm start # for production
```
or 
```
npm run dev # for development
```
